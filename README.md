# EditableNotepad

EditableNotepad is a small text editor built with Java Swing. It provides basic notepad functionality (open, edit, save) and demonstrates a lightweight desktop UI using standard Swing components.

## Features
- File: New, Open, Save, Save As, Exit
- Edit: Cut, Copy, Paste, Delete, Select All
- Tracks unsaved changes and prompts to save before closing
- About dialog with basic app information

## Requirements
- Java 8 or newer (JDK)

## Build & Run
From the project root, compile and run the main class:

```
javac EditableNotepadEx.java
java EditableNotepadEx
```

The main source file is [EditableNotepadEx.java](EditableNotepadEx.java) which contains the GUI and core functionality.

## Notes
- No external libraries required — uses Java Swing (part of the JDK).
- The application is intentionally simple and can be extended with additional features (search, undo/redo, preferences).

Contributions and improvements are welcome.
